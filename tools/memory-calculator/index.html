로컬 AI 메모리 계산기

모델 가중치만이 아니라 KV 캐시, 런타임 버퍼, RAG 인덱스, 데이터 working set, 이미지·오디오 파이프라인, 동시성 및 파인튜닝 상태를 함께 계산합니다.

자동 탐지는 근사값입니다

navigator.deviceMemory는 일부 브라우저에서만 제공되는 반올림·상한 적용 RAM 값입니다. 표준 웹 API는 정확한 총 VRAM을 제공하지 않습니다. 계산기는 브라우저가 비표준 WebGPU memoryHeaps를 공개한 경우 그 값을 표시하고, 그 외에는 GPU 이름과 알려진 제품 사양을 대조합니다. 변형이 여러 개인 장치는 후보를 제시하며 수동 입력값이 최종 기준입니다.

탐지·계산·저장은 이 브라우저 안에서만 수행됩니다. 외부 API 호출과 텔레메트리는 없습니다.
